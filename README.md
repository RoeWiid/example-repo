Project Name: finance_calculators.py
Created code that can be used as a finance calculator
The code allows the user to access two different financial calculators: 
an investment calculator and a home loan repayment calculator.

User will be able to select either calculator.

1) If the user selects “investment”, ask the user to input:
○ The amount of money that they are depositing.
○ The interest rate (as a percentage). 
○ The number of years they plan on investing.
○ The user can then select between “simple” or “compound” interest, and store this in a variable called interest.

2)If the user selects “bond”, ask the user to input:
○ The present value of the house, e.g., 100 000.
○ The interest rate, e.g., 7.
○ The number of months they plan to take to repay the bond, e.g., 120.
○ The calculator will then calculate how much money the user will have to repay each month and output the answer.
